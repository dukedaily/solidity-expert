# 第5.1章：Foundry 框架

> 小白入门：https://github.com/dukedaily/solidity-expert ，欢迎star转发，文末加V入群。
>
> 职场进阶: https://dukeweb3.com

Foundry 是 Solidity 生态的主流测试与部署框架，由 Paradigm 开发，用 Rust 编写。速度极快（测试秒级）、Fork / Fuzz / Invariant 原生支持。本章按工具链逐步拆解，适合已掌握 Solidity 基础、想从 Hardhat 迁移或从零上手的开发者。
